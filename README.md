# 用户体验
## 视觉

* 注意排版的，不要让信息过于拥挤
* 统一视觉设计规范,提升可识别性和用户学习成本
* 重点内容或者可交互物的视觉突显(动画或颜色)
* 合理搭配内容,色块搭配 > 大量线框
* 表单操作实时逐行效验,不要统一效验,尽量减少必填项
* 表单尽可能精简,表单项内容提示
* 行高为字号的1.5-2倍
* 盒子内边距
* 前端还原视觉稿
* 内容锚点标记突显(如果有)
* 加载资源优化
* 异步操作Loading
* 多内容文本域摘要
* 合并相似的功能
* 内容区域分类清晰

## 交互操作

* 操作步骤最小化
* 下拉框增加默认数据减少用户操作
* 敏感操作出确认框提示
* 实时搜索框节流
* 带数据交互按钮防抖
* 所有操作都要给出反馈,让用户知道操作已生效
